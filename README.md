# spring-boot-app

A simple sample application Spring Boot

## Requires

- Java (>= 1.8)
- Apache Maven (>= 3.3.9)
- Docker (>= 1.11.2)
- Docker Machine (>= 0.7.0)
- Docker Compose (>= 1.8.0)

## Database
Creating a [Docker](https://www.docker.com/) container image for [PostgreSQL](http://postgresql.org/)

## Build
- mvn clean package

## Run
- mvn spring-boot:run
