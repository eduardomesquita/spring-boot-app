# spring-boot-app

A simple sample application Spring Boot

<h1> Installation </h1> 

<h2> Requires </h2>
- Java (>= 1.8)
- Apache Maven (>= 3.3.9)

  Docker
- Docker (>= 1.11.2)
- Docker Machine (>= 0.7.0)
- Docker Compose (>= 1.8.0)

<h1> Build </h1>
- mvn clean package

<h1> Run </h1>
- mvn spring-boot:run
